# Spring Boot入門 - プロジェクト構成

## Spring Bootとは

Spring BootはJavaベースのWebアプリケーションフレームワークです。

## プロジェクト構成

```
src/
├── main/
│   ├── java/com/example/
│   │   ├── controller/   # REST API
│   │   ├── service/      # ビジネスロジック
│   │   ├── repository/   # データアクセス
│   │   ├── entity/       # エンティティ
│   │   └── dto/          # データ転送オブジェクト
│   └── resources/
│       ├── application.yml
│       └── db/migration/
└── test/
```

## 最初のController

```java
@RestController
@RequestMapping("/api/hello")
public class HelloController {

    @GetMapping
    public String hello() {
        return "Hello, Spring Boot!";
    }
}
```

## 依存性注入

```java
@Service
@RequiredArgsConstructor
public class UserService {
    private final UserRepository userRepository;

    public User findById(Long id) {
        return userRepository.findById(id)
            .orElseThrow(() -> new RuntimeException("Not found"));
    }
}
```
