# Java入門 - 基本構文

## 変数と型

Javaは静的型付け言語です。変数を宣言するときは型を指定します。

```java
int number = 42;
String message = "Hello, World!";
boolean isActive = true;
double price = 19.99;
```

## 条件分岐

```java
if (score >= 80) {
    System.out.println("優秀です");
} else if (score >= 60) {
    System.out.println("合格です");
} else {
    System.out.println("再挑戦してください");
}
```

## ループ

```java
// for文
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// 拡張for文
List<String> names = List.of("Alice", "Bob", "Charlie");
for (String name : names) {
    System.out.println(name);
}
```

## まとめ

- Javaは静的型付けで安全なコードが書ける
- 条件分岐とループで制御フローを作る
- コレクションを活用しよう
